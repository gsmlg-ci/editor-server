# editor-server

[![Build](https://github.com/gsmlg-ci/editor-server/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/editor-server/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/editor-server`
- `ghcr.io/gsmlg-dev/editor-server`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/editor-server:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/editor-server:latest
```

## Build

```bash
docker build -t gsmlg/editor-server .
```
